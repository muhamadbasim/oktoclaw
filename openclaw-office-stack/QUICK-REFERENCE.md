# QUICK-REFERENCE.md - One-line guide for all 10 office agents

Pakai file ini kalau kamu cuma butuh jawaban cepat: **agent ini buat apa?**

## 10 office agents in one line

- `office-main` → pecah request campuran jadi workstream dan arahkan ke agent yang tepat
- `minutes` → ubah meeting/transcript jadi notulen, keputusan, dan action items
- `inbox` → triage email/thread, tangkap ask penting, dan draft balasan
- `project` → rapikan task, owner, due date, blocker, dan status proyek
- `report` → ubah KPI/angka jadi insight, risiko, dan tindakan lanjut
- `docs` → ubah catatan mentah jadi SOP, handover, atau dokumentasi rapi
- `support` → triage issue customer, tentukan severity, dan buat first response
- `sales` → buat lead brief, outreach draft, dan summary pipeline/follow-up
- `competitor` → ringkas perubahan kompetitor dan jelaskan dampaknya ke bisnis
- `code-review` → sorot risk teknis, missing tests, dan review summary dari PR/diff

## Fast choose

- bingung mulai dari mana → `office-main`
- habis meeting → `minutes`
- ada email masuk → `inbox`
- proyek mulai berantakan → `project`
- banyak angka/KPI → `report`
- butuh SOP/dokumen → `docs`
- ada tiket/customer issue → `support`
- ada prospect/lead → `sales`
- mau pantau kompetitor → `competitor`
- mau review code/PR → `code-review`

## Recommended order for new users

1. `office-main`
2. `minutes`
3. `inbox`
4. `project`
5. sisanya sesuai kebutuhan
